# NIHONGO.GO

Phase 1 foundation for a pixel-style Japanese travel phrase PWA.

## Commands

```bash
npm run content:generate
npm run content:check
npm run dev
npm run lint
npm run build
```

## Phase 1 scope

- Next.js 14 App Router scaffold
- `next-pwa` setup with offline fallback
- scene-split static JSON content
- fixed `localStorage` schema for `ReviewItem`
- unlock logic utility
- native `<ruby>` rendering only
- iOS Safari speech and IME test bench
